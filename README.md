# Machine Learning Projects Portfolio

## 📌 Overview
This repository contains multiple machine learning projects completed during my internship and learning journey. These projects cover different domains including time-series forecasting, NLP, model evaluation, and synthetic data generation.

---

## 📁 Projects Included

### 1. Sales Forecasting System Using Time-Series Analysis
- Predicts future sales using regression and time-series models
- Techniques used: Linear Regression, Exponential Smoothing

### 2. News Topic Classification System
- Classifies news articles into categories using NLP
- Models used: Naive Bayes, SVM
- Feature extraction: TF-IDF

### 3. ML Model Comparison and Evaluation System
- Compares multiple ML models on structured dataset
- Models: Logistic Regression, Decision Tree, Random Forest, SVM, KNN, Naive Bayes

### 4. Adaptive Synthetic Data Augmentation Toolkit
- Fraud detection system using synthetic data generation
- Uses CTGAN to handle class imbalance
- Includes iterative training for model improvement
